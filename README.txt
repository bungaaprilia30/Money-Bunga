Money Bunga - GitHub Pages Edition

Panduan singkat:
1. Unggah seluruh isi folder 'Money-Bunga' (extract) ke repository GitHub bernama:
   Money-Bunga
   (pastikan file berada di root repository)

2. Struktur file yang harus ada di root repo:
   - index.html
   - manifest.json
   - service-worker.js
   - assets/icon.png

3. Aktifkan GitHub Pages:
   Settings -> Pages
   Source: Branch 'main' (atau 'master') -> Folder '/' (root)
   Simpan, tunggu ~1 menit.

4. Buka:
   https://{USERNAME}.github.io/Money-Bunga/
   (ganti {USERNAME} dengan username GitHub kamu)

5. Di HP, buka link tersebut -> pilih 'Add to Home screen' atau 'Install' di browser untuk menambahkan ikon.

Catatan:
- Service worker bekerja di GitHub Pages (HTTPS). Jika tidak muncul, tunggu beberapa menit dan bersihkan cache browser.
- Jika ingin mengganti ikon, ganti file assets/icon.png dan commit ulang.
