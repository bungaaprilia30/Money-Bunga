Money Bunga - GitHub Pages (Full Version)

Panduan singkat:
1. Extract isi ZIP dan upload seluruh folder 'Money-Bunga' ke repository GitHub kamu
   Pastikan file berada di root repo:
     - index.html
     - manifest.json
     - service-worker.js
     - assets/icon.png

2. Aktifkan GitHub Pages:
   - Settings -> Pages
   - Source: Branch 'main' -> Folder '/' (root)
   - Save. Tunggu ~1 menit.

3. Buka: https://<username>.github.io/Money-Bunga/
   Di HP, pilih 'Add to Home screen' atau 'Install' untuk menambahkan aplikasi.

Catatan: Jika tampilan tidak berubah setelah upload, coba bersihkan cache browser atau tunggu 1-2 menit.
